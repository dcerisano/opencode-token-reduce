# Suggested Commands

| Command | Purpose |
|---|---|
| `opencode` | Launch OpenCode |
| `/dcp context` | DCP: show token usage breakdown |
| `/dcp stats` | DCP: show pruning statistics |
| `/dcp sweep` | DCP: prune tools since last user message |
| `opencode --prompt "startup"` | Start with startup prompt |
| `gh repo create <name> --template dcerisano/opencode-token-reduce --public --clone` | Create new project from template |
| `serena memories check` | Check for stale memories (from project root) |

Serena is started automatically by OpenCode via `opencode.json` MCP config, not manually.
