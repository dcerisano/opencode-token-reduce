# Suggested Commands

## Template Usage
- `opencode --prompt "startup"` — start a session with the template
- `opencode --prompt "/migrate /path/to/target"` — migrate template files into existing project

## Memory Management
- `serena memories check` — verify memory integrity from project root

## Project Creation (GitHub)
- `gh repo create my-project --template dcerisano/opencode-token-reduce --public --clone` — bootstrap new project from template
