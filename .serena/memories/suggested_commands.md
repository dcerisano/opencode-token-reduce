# Suggested Commands

## Startup
```bash
# Launch OpenCode with this template's startup prompt
opencode --prompt "startup"
```

## Migration (apply template to existing project)
```bash
# From within a target project:
opencode --prompt "/migrate /path/to/target/project"
```
See `mem:migration` for the detailed `/migrate` workflow.

## No build/test/lint
This is a configuration template with zero application code. There are no build, test, or lint commands to run.
