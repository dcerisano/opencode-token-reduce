---
name: detect-languages
description: Detect serena languages in the project by scanning source directories. Writes `.serena/project.local.yml` and a cache signature. Reports whether languages changed since last run. Called by bootstrap — do not use directly.
---

Detects programming languages in the project for serena LSP configuration. Replaces the `detect-languages.sh` script with model-level operations (no bash calls).

### Steps

1. **Find all source files with known extensions** at the project root and under all directories (including dot-directories). The glob tool does not descend into hidden directories, so use Bash `find` for this step:

   ```bash
   find . -type f -not -path './.git/*' -not -path '*/node_modules/*' \( -name '*.sh' -o -name '*.java' -o -name '*.py' -o -name '*.ts' -o -name '*.tsx' -o -name '*.js' -o -name '*.jsx' -o -name '*.md' -o -name '*.json' -o -name '*.yaml' -o -name '*.yml' -o -name '*.go' -o -name '*.rs' -o -name '*.rb' -o -name '*.kt' -o -name '*.css' -o -name '*.scss' -o -name '*.php' \)
   ```

2. **Map extensions to serena language names** (deduplicate):

   | Extension(s) | Language    |
   |--------------|-------------|
   | sh           | bash        |
   | java         | java        |
   | py           | python      |
   | ts, tsx, js, jsx | typescript |
   | md           | markdown    |
   | json         | json        |
   | yaml, yml    | yaml        |
   | go           | go          |
   | rs           | rust        |
   | rb           | ruby        |
   | kt           | kotlin      |
   | css, scss    | scss        |
   | php          | php         |

3. **Sort** alphabetically.

4. **Read** `.serena/project.local.yml` (previous state).

5. **Compare**: if the sorted language list matches the previous content (ignoring yaml header), report "No language changes." and stop.

6. **Write** `.serena/project.local.yml` with:
   ```yaml
   # Auto-generated by detect-languages skill
   languages:
     - <sorted lang 1>
     - <sorted lang 2>
     ...
   ```

7. **Write** `.serena/.detected-languages.sig` with the sorted language list joined by newlines (plain text).

8. **Report**: "Detected languages: <list>" and if changed, "New languages detected — restart opencode for serena LSPs to activate."
